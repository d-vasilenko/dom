# 📘 DOM + JavaScript: От Основ до Продакшена
> Менторский курс с глубокой теорией, практикой и строгим контролем качества.  
> Никакой воды. Только современные стандарты, реальные UI-паттерны, архитектура и отладка.

---

## 🎯 Философия и формат
- **Принцип работы:** `Теория → Практика → Код-ревью → Переход только после ✅ разобрался`
- **Уровень:** от уверенных основ до senior-подхода к работе с DOM без фреймворков
- **Фокус:** как браузер работает под капотом, спецификации, производительность, безопасность, a11y, современные API, отладка, тесты
- **Моя роль:** ментор-наставник. Даю сжатую, но глубокую теорию, разбираю твой код построчно, указываю на антипаттерны, даю ссылки на спеки, не пропускаю дальше без полного понимания
- **Твоя роль:** писать код, тестировать в браузере, задавать вопросы, рефлексировать, отправлять отчёт по шаблону
